# Google Ads Campaign — River Mountain Systems
**Budget:** $20/day ($600/mo) · **Goal:** 10–20 clicks/day, 1–3 form fills or calls/week
**Duration:** Run 2 weeks minimum before judging. Pause any ad group spending >$50 with 0 conversions.

---

## Account Setup (do this first)

1. **Conversion actions to create** (Tools → Conversions):
   - Phone call from ads (auto-created when you link the phone number)
   - Phone call on website — tag fires on `tel:` clicks (already wired in the site)
   - Form submission — tag fires on `form_submit` event (already wired)
   - Set all to "Primary" conversion goal

2. **Location targeting:** Vancouver WA + 30-mile radius. OR manual cities: Vancouver, Camas, Washougal, Battle Ground, Ridgefield, La Center, Portland OR, Gresham, Beaverton, Hillsboro, Tigard, Lake Oswego, Tualatin

3. **Schedule:** Mon–Fri 7am–7pm PT, Sat 8am–2pm PT (align with Justin's availability)

4. **Bidding:** Start with "Maximize clicks" for 2 weeks (gather data). Switch to "Target CPA" once you have 15+ conversions.

5. **Ad rotation:** "Optimize" (Google picks the best performing ad)

## Weekly Optimization Checklist

Run this every Monday:

- [ ] Check Search Terms report — add irrelevant queries as negatives
- [ ] Check Impression Share — if <50%, either raise budget or tighten match types
- [ ] Check CTR by ad — pause headlines with CTR <2% after 100+ impressions
- [ ] Check conversion rate by landing page — if IT hub is converting and WMS isn't, shift budget
- [ ] Check "Top vs Other" position — aim for "Absolute Top" on brand terms
- [ ] Adjust bids by device — if mobile converts worse, reduce mobile bid modifier by 20%
- [ ] Add high-performing search terms as exact match keywords

## First 2-Week Goals

| Metric | Target |
|---|---|
| Impressions | 1,000–3,000/week |
| CTR | >3% (industry avg is 2%) |
| Clicks | 30–60/week |
| CPC | <$8 |
| Calls or form fills | 1–3/week |
| Cost per lead | <$200 |

If CPC is >$15 on day 3, pause and check: are the keywords too broad? Is Quality Score low? Fix the landing page first.
